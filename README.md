# Components Included
  - Core interfaces, Autowiring DI, declarative singletons and auto injections
  - Tokenizer, static code analysis
  - Debug and Dump
  - FileManager
  - Security layer (NIST RBAC)
  - Reactor, code scaffolding

# Tests
To run tests execute following commands
```
composer install
phpunit
```
